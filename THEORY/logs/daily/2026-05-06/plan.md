### §2.2.1 Sub-file `02a_unordered_configuration_formalism.md` (~5-7 KB; ~2-2.5h budget 의 ~1h)

- §1.1 **Symmetric group action** definition. $S_K = S_{K_{\mathrm{field}}}$ 가 K개 slot index 의 permutation. Action on configuration $\mathbf{u} = (u^{(1)}, \ldots, u^{(K)}) \in \Sigma_M^K$ via $\sigma \cdot \mathbf{u} = (u^{(\sigma^{-1}(1))}, \ldots, u^{(\sigma^{-1}(K))})$.
- §1.2 **Quotient construction**. $\widetilde{\widetilde\Sigma}^K_M := \Sigma_M^K / S_K$ 의 quotient topology. Equivalence class $[\mathbf{u}]$ 의 explicit 정의.
- §1.3 **Stratification analysis**. Orbit structure: open stratum (free orbit, $|S_K \cdot \mathbf{u}| = K!$) vs symmetric strata (stabilizer non-trivial; e.g., 2개 slot 의 $u^{(j)} = u^{(k)}$ identical 의 경우 $\mathbb{Z}_2$ stabilizer). Symmetric strata 의 codimension count.
- §1.4 **Dimension count**. Open stratum 의 dimension = $K \cdot (n-1)$ (per-slot Σ_m 의 dim = n-1, K slots). Symmetric strata dimension reduction by orbit structure.
- §1.5 **Worked example**: $T^2_4$ (n=16) graph, $K=2$. Open stratum dim = $2 \cdot 15 = 30$. Symmetric stratum (mirror $u^{(1)} = u^{(2)}$) dim = 15, codim 15. Explicit configuration sample + class representative selection rule.

### §2.2.2 Sub-file `02b_reduction_map_pi.md` (~3-4 KB; ~30-40 min)

- §2.1 **Explicit construction**. $\pi : \Sigma_M^K \to \widetilde{\widetilde\Sigma}^K_M$ canonical projection $\pi(\mathbf{u}) = [\mathbf{u}]$.
- §2.2 **Continuity**. Quotient topology 정의에 의해 π continuous.
- §2.3 **Local sections**. 각 open stratum 위에서 ordered representative 선택 rule (e.g., lexicographic on $\sum u^{(j)}$ values; tie-break by slot index). Local section $s : U \to \Sigma_M^K$ continuous on each open stratum. Symmetric strata 위에서는 multi-valued (representative ambiguity).
- §2.4 **Fiber structure**. $\pi^{-1}([\mathbf{u}])$ = $S_K \cdot \mathbf{u}$ orbit. Fiber dimension = 0 generically (discrete orbit), 양수 only when stabilizer 가 continuous (이 case 발생 안함, $S_K$ finite).

### §2.2.3 Sub-file `02c_minimization_principle_unordered.md` (~4-5 KB; ~30-40 min, 부분 sketch acceptable)

- §3.1 **$S_K$-equivariance check on $\mathcal{E}$**. Energy $\mathcal{E}(\mathbf{u}) = \lambda_{\mathrm{cl}} \mathcal{E}_{\mathrm{cl}} + \lambda_{\mathrm{sep}} \mathcal{E}_{\mathrm{sep}} + \lambda_{\mathrm{bd}} \mathcal{E}_{\mathrm{bd}} + \lambda_{\mathrm{tr}} \mathcal{E}_{\mathrm{tr}} + \lambda_{\mathrm{rep}} \sum_{j<k} \langle u^{(j)}, u^{(k)}\rangle$. Inter-field repulsion $\lambda_{\mathrm{rep}}$ term 의 $S_K$-invariance: $\langle u^{(\sigma^{-1}(j))}, u^{(\sigma^{-1}(k))}\rangle$ 의 sum over $j<k$ pairs = original sum (symmetric polynomial invariant). 즉 $\mathcal{E}(\sigma \cdot \mathbf{u}) = \mathcal{E}(\mathbf{u})$. Other terms 도 동일하게 $S_K$-invariant.
- §3.2 **Induced energy on $\widetilde{\widetilde\Sigma}^K_M$**. $\widetilde{\mathcal{E}}([\mathbf{u}]) := \mathcal{E}(\mathbf{u})$ well-defined by §3.1.
