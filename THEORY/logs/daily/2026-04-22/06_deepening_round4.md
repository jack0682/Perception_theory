## §1. Setup and strategy

### 1.1 Why these two cases

Round 3 showed: on 2D square grid with free BC and $\mathrm{Aut}(G) = D_4$, the first Fiedler pitchfork produces an **axis-aligned 4-point orbit** of minimizers with ratio $A_2/A_1 = 4$ > 2 (anisotropic). This used:
- Fiedler subspace is a 2D irreducible representation of $D_4$.
- Eigenvectors are pure cosines aligned with axes (free BC).
- Integrals $I_4 = 3/2$, $K = 1$, $J = 0$ combined to give orbit selection.

Non-D4 graph classes change the setup along two axes:
- **Dimension of Fiedler eigenspace.** Translation symmetry creates degenerate $\pm k$ modes (sine + cosine), doubling the eigenspace per translation direction.
- **Isotropy group structure.** On cycle: $D_n$ acts by rotation. On torus: $(\mathbb Z_L)^2$ acts by translation and $D_4$ acts by rotation/reflection.

**Strategy.** Same Landau-reduction structure as Round 3:
1. Identify Fiedler eigenspace (dimension $d_F$) and its $\mathrm{Aut}(G)$-action.
2. Compute quartic integrals $\int \phi_i \phi_j \phi_k \phi_l$ (restricted by parity).
3. Assemble reduced Lyapunov $F$ on $\mathbb R^{d_F}$.
4. Classify orbits via isotropy subgroups; compute $F$ on each.
5. Compare energies and stabilities.

At $c = 1/2$, all cubic-in-$v$ terms vanish (double-well symmetry), so the reduction is purely quartic.

### 1.2 Normalization conventions

Use $L^2$-normalized eigenvectors: $\int \phi_k^2 = 1$ (continuum) or $\sum_i \phi_k(i)^2 = 1$ (lattice). At $c = 1/2$ the double-well $W(c + v) = 1/16 - v^2/2 + v^4$ has quartic coefficient $W^{(4)}(c)/24 = 1$ (since $W^{(4)}(c) = 24$). Absorbing this into $\Lambda := \beta_{\mathrm{bd}}$, the reduced energy is
$$F(\vec v) = \tfrac{\mu}{2}\|\vec v\|^2 + \Lambda \int (\text{expansion of } v^4).$$

This matches Round 3 conventions (§3.3, `symmetry_moduli.md`).

### 1.3 Discrete-lattice exactness

For periodic graphs with period $L$ or $n$, eigenvectors are **exactly** $\cos(2\pi k \cdot/n)$, $\sin(2\pi k \cdot /n)$. For quartic integrals, we use the identity
$$\sum_{k=0}^{n-1}\cos^4(2\pi k/n) = \frac{3n}{8},\qquad (n \geq 5),$$
and analogous formulas for other monomials. So integrals are **exact for $n \geq 5$** (no $O(1/n^2)$ lattice correction). This is a stronger regularity than Round 3's free-BC case (where the boundary cosines give exactness only in the continuum limit).

## §2. 1D cycle $C_n$

### 2.1 Fiedler subspace

Vertex set $X = \{0, 1, \ldots, n-1\}$, edges $\{i, i+1 \bmod n\}$. Laplacian eigenvalues $\lambda_k = 4\sin^2(\pi k/n)$ for $k = 0, 1, \ldots, n-1$. Fiedler eigenvalue $\lambda_2 = 4\sin^2(\pi/n)$ is **2-fold degenerate** (from $k = 1$ and $k = n-1$ giving the same $\lambda$).

**Real-valued orthonormal basis:**
$$\phi_1(j) = \sqrt{2/n}\cos(2\pi j/n),\qquad \phi_2(j) = \sqrt{2/n}\sin(2\pi j/n).$$

### 2.2 $\mathrm{Aut}(C_n) = D_n$ action on Fiedler subspace

- Translation $T_m : j \mapsto j + m \pmod n$ acts on $(\phi_1, \phi_2)$ as rotation by angle $\theta = 2\pi m/n$:
  $(\phi_1, \phi_2) \mapsto (\phi_1 \cos\theta + \phi_2 \sin\theta, -\phi_1 \sin\theta + \phi_2 \cos\theta)$.
- Reflection $R: j \mapsto -j \pmod n$: $(\phi_1, \phi_2) \mapsto (\phi_1, -\phi_2)$.

So on the 2D Fiedler subspace, $D_n$ acts as the dihedral subgroup of $O(2)$ generated by rotation $2\pi/n$ and reflection. For large $n$, $D_n \subset O(2)$ becomes dense.

### 2.3 Quartic integrals (exact for $n \geq 5$)

Parameterize $v = a\phi_1 + b\phi_2$.

**Diagonal integrals** $I_4 := \sum_j \phi_1(j)^4 = \sum_j \phi_2(j)^4$:
$$I_4 = (2/n)^2 \sum_{j=0}^{n-1}\cos^4(2\pi j/n) = (4/n^2) \cdot (3n/8) = \frac{3}{2n}.$$
Wait — recheck normalization. $\phi_1(j) = \sqrt{2/n}\cos(2\pi j/n)$, so $\phi_1(j)^4 = (2/n)^2 \cos^4 = (4/n^2)\cos^4$. Sum: $(4/n^2)(3n/8) = 3/(2n)$.

Hmm, this seems to have $1/n$ scaling — but that's because the sum is over $n$ sites, each with weight $\sim 1/n$ from normalization. Let me re-parameterize: use the **expectation** $\langle \phi^4 \rangle := (1/n)\sum_j \phi^4(j)$ instead.

Actually, to match Round 3's continuum convention (where $\int \phi^4 dx$ over unit interval $= 3/2$), we should think of $\phi$ as a **density**: the discrete sum $\sum_j \phi^4(j)$ corresponds to $\int \phi^4 dx$ with lattice spacing $\Delta x = 1/n$... but with $\phi$ L²-normalized on the total mass, the factor of $1/n$ in the expectation vanishes.

**Clean convention.** Let $\phi_1(j) = \sqrt 2 \cos(2\pi j/n)$, $\phi_2(j) = \sqrt 2 \sin(2\pi j/n)$ (without $1/\sqrt n$ factor). Then $\|\phi_k\|^2 = \sum_j 2\cos^2(2\pi j/n) = n$ (normalized to total-mass). Dividing by $n$ gives $\langle \phi_k^2 \rangle = 1$ as expected.

For the quartic form, use $\int \phi^4 := (1/n)\sum_j \phi^4(j)$:
$$I_4 = (1/n) \sum (\sqrt 2)^4 \cos^4 = (4/n)(3n/8) = 3/2.$$ 

Same as 2D square grid result. Good.

$$K := (1/n)\sum \phi_1^2 \phi_2^2 = (1/n)\sum 4\cos^2\sin^2 = (1/n)\sum \sin^2(4\pi j/n).$$
For $n \geq 5$: $\sum_{j=0}^{n-1}\sin^2(4\pi j/n) = n/2$. So $K = 1/2$.

$$J := (1/n)\sum \phi_1^3 \phi_2 = (1/n)\sum 4\cos^3\sin \cdot dx.$$
Use $\cos^3\sin = -\frac{1}{4}\frac{d}{d\theta}\cos^4$, which telescopes over a full period: $\sum_{j=0}^{n-1}\cos^3(2\pi j/n)\sin(2\pi j/n) = 0$. So $J = 0$.

### 2.4 Reduced Lyapunov function

$$F(a, b) = \tfrac{\mu}{2}(a^2 + b^2) + \Lambda [I_4(a^4 + b^4) + 6K a^2 b^2]$$
$$= \tfrac{\mu}{2}(a^2 + b^2) + \Lambda [\tfrac{3}{2}(a^4 + b^4) + 3 a^2 b^2].$$

With $A_1 := \tfrac{3}{2}\Lambda$, $A_2 := 3\Lambda$: **ratio $A_2/A_1 = 2$** — the $O(2)$-invariant case.

**Verification.** $A_1(a^4 + b^4) + A_2 a^2 b^2 = \tfrac{3}{2}\Lambda(a^4+b^4) + 3\Lambda a^2 b^2 = \tfrac{3}{2}\Lambda(a^4 + b^4 + 2a^2 b^2) = \tfrac{3}{2}\Lambda(a^2 + b^2)^2$. Purely radial. ✓

### 2.5 Critical manifold

$F(r) = \tfrac{\mu}{2} r^2 + \tfrac{3}{2}\Lambda r^4$ where $r^2 = a^2 + b^2$.
$F'(r) = \mu r + 6\Lambda r^3 = 0 \Rightarrow r^2 = -\mu/(6\Lambda)$ (for $\mu < 0, \Lambda > 0$).
Minimum value $F_{\min} = -\mu^2/(24\Lambda)$.

**Critical set: a 1-dim circle** $\{(a, b) : a^2 + b^2 = -\mu/(6\Lambda)\}$ in the Fiedler 2D subspace.

### 2.6 Hessian and Goldstone direction

At any point on the circle (say $(a, b) = (R, 0)$, $R^2 = -\mu/(6\Lambda)$):
- $F_{aa} = \mu + 6\Lambda(3a^2 + b^2) = \mu + 18\Lambda R^2 = \mu - 3\mu = -2\mu > 0$ (radial, stable).
- $F_{bb} = \mu + 6\Lambda(a^2 + 3b^2) = \mu + 6\Lambda R^2 = 0$ (angular, **Goldstone** from $O(2)$).
- $F_{ab} = 12\Lambda a b = 0$.

So the Hessian at each circle point has signature $\{-2\mu, 0\}$ — one stable and one marginal.

### 2.7 $D_n$ breaks $O(2)$ to discrete rotations

The circle is not a true continuous family of the full problem — it is a family of quartic-level minimizers. Higher-order terms (sextic and beyond) are NOT $O(2)$-invariant because $D_n$ (the true symmetry) does not extend to $O(2)$ for finite $n$. Leading anisotropy: a $\cos(n\theta)$ term in the angular part of $F$ at order $2n$ or higher (depends on discretization). 

**Consequence for $\mathcal{M}_1$:**
- **In continuum limit ($n \to \infty$)**: $\mathcal{M}_1 \cong S^1/D_\infty = [0, \pi]$ (interval, 1-dim moduli).
- **At finite $n$**: circle is broken into $2n$ standing-wave orbits at high order; $\mathcal{M}_1$ is a discrete set of size $2n/|D_n| = 1$ after quotienting (all standing waves are equivalent under $D_n$).
- **Practical observation**: Any local minimum at first pitchfork on $C_n$ is a single-bump standing wave $\sim \cos(2\pi(j - j_0)/n)$ with center $j_0 \in \{0, 1, \ldots, n-1\}$. All such are $D_n$-equivalent.

**Summary (Round 4 result for $C_n$):**

> **$C_n$ First-Pitchfork Theorem.** On the 1D cycle $C_n$ ($n \geq 5$) at $c = 1/2$, the reduced Lyapunov function at first Fiedler pitchfork is $F(r) = \tfrac{\mu}{2}r^2 + \tfrac{3}{2}\Lambda r^4$ (purely radial), giving a 1-dim circle of degenerate minima with radius $R = \sqrt{-\mu/(6\Lambda)}$. The circle is broken to $2n$ discrete standing-wave orbits at sextic or higher order, but all $2n$ are $D_n$-equivalent. Therefore $|\mathcal{M}_1(C_n)| = 1$ at $\beta$ just above $\beta_{\mathrm{crit}}^{(2)}$, with **orbit size $= n$** (much larger than $D_4$'s size 4 on square grid).

**Category: Cat A** — exact discrete integrals (no $O(1/n^2)$ correction), elementary algebra.

## §3. 2D torus $T^2 = C_L \times C_L$

### 3.1 Fiedler subspace (4-dim)

Laplacian eigenvalues $\lambda_{k_1, k_2} = 4\sin^2(\pi k_1/L) + 4\sin^2(\pi k_2/L)$. Fiedler eigenvalue $\lambda_2 = 4\sin^2(\pi/L)$ is **4-fold degenerate** from $(k_1, k_2) \in \{(1,0), (L-1, 0), (0, 1), (0, L-1)\}$.

**Real-valued orthonormal basis** (using $(1/L^2)\sum \phi^2 = 1$):
$$\phi_X^c(i,j) = \sqrt 2 \cos(2\pi i/L),\quad \phi_X^s = \sqrt 2 \sin(2\pi i/L),$$
$$\phi_Y^c = \sqrt 2\cos(2\pi j/L),\quad \phi_Y^s = \sqrt 2\sin(2\pi j/L).$$

### 3.2 $\mathrm{Aut}(T^2) = D_4 \ltimes (\mathbb Z_L)^2$ action on Fiedler 4-subspace

- **X-translation by $m$:** rotates $(\phi_X^c, \phi_X^s)$ by $2\pi m/L$; leaves $(\phi_Y^c, \phi_Y^s)$ fixed.
- **Y-translation** analogously.
- **D_4 action** (90° rotation of lattice, reflections): swaps X-block with Y-block, applies reflection within each block.

### 3.3 Quartic integrals (exact for $L \geq 5$)

Using the cycle identities from §2.3, the non-zero quartic integrals (those with even power in each of the four basis indices) are:

- **Fourth powers:** $\langle (\phi_X^c)^4 \rangle = \langle (\phi_X^s)^4 \rangle = \langle (\phi_Y^c)^4 \rangle = \langle (\phi_Y^s)^4 \rangle = 3/2$.
- **Within-block crosses:** $\langle (\phi_X^c)^2 (\phi_X^s)^2 \rangle = \langle (\phi_Y^c)^2 (\phi_Y^s)^2 \rangle = 1/2$.
- **Between-block crosses:** $\langle (\phi_X^c)^2 (\phi_Y^c)^2 \rangle = 1$, and analogously for all 4 X-Y pairs (each $= 1$).

All odd-index terms (e.g., $\phi_X^c \phi_X^s \phi_Y^c \phi_Y^s$, $(\phi_X^c)^3 \phi_Y^c$) vanish by the cycle-identity for odd powers.

### 3.4 Reduced Lyapunov function

Parameterize $v = a\phi_X^c + b\phi_X^s + c'\phi_Y^c + d\phi_Y^s$. Define
$$r_1^2 := a^2 + b^2 \quad (\text{X-amplitude}),\qquad r_2^2 := c'^2 + d^2 \quad (\text{Y-amplitude}).$$

Assembling:
$$\int v^4 = \tfrac{3}{2}(a^4 + b^4 + c'^4 + d^4) + 3(a^2 b^2 + c'^2 d^2) + 6 r_1^2 r_2^2.$$

Using $a^4 + b^4 = r_1^4 - 2a^2 b^2$ (and similarly for Y-block):
$$\int v^4 = \tfrac{3}{2}(r_1^4 + r_2^4) + 6 r_1^2 r_2^2 = \tfrac{3}{2}(r_1^2 + r_2^2)^2 + 3 r_1^2 r_2^2.$$

**Reduced Lyapunov:**
$$F(r_1, r_2) = \tfrac{\mu}{2}(r_1^2 + r_2^2) + \Lambda\!\left[\tfrac{3}{2}(r_1^2 + r_2^2)^2 + 3 r_1^2 r_2^2\right].$$

**Structural observation.** Within each block separately, the structure is $O(2)$-invariant (inherited from each 1D cycle factor). The **inter-block coupling** $+3 r_1^2 r_2^2$ is the only isotropy-breaking term at quartic order.

### 3.5 Critical points and energies

Parameterize by $(r_1, r_2) \in \mathbb R_{\geq 0}^2$.

$\partial F/\partial r_1 = r_1[\mu + 6\Lambda(r_1^2 + r_2^2) + 6\Lambda r_2^2] = r_1[\mu + 6\Lambda(r_1^2 + 2 r_2^2)]$.
$\partial F/\partial r_2 = r_2[\mu + 6\Lambda(2 r_1^2 + r_2^2)]$.

Three solution families (beyond trivial $(0, 0)$):

**(a) Pure-X orbit** $(r_1, r_2) = (R, 0)$: $\mu + 6\Lambda R^2 = 0 \Rightarrow R^2 = -\mu/(6\Lambda)$.
$F_X = \tfrac{\mu}{2}R^2 + \tfrac{3\Lambda}{2}R^4 = -\mu^2/(24\Lambda)$.

**(b) Pure-Y orbit** $(0, R)$: same by $X \leftrightarrow Y$ symmetry, $F_Y = -\mu^2/(24\Lambda)$.

**(c) Diagonal orbit** $(s, s)$: $\mu + 18\Lambda s^2 = 0 \Rightarrow s^2 = -\mu/(18\Lambda)$.
$F_\Delta = \mu s^2 + 9\Lambda s^4 = \mu \cdot (-\mu/(18\Lambda)) + 9\Lambda \cdot \mu^2/(324\Lambda^2) = -\mu^2/(18\Lambda) + \mu^2/(36\Lambda) = -\mu^2/(36\Lambda)$.

**Comparison:** $|F_X| = \mu^2/(24\Lambda) > \mu^2/(36\Lambda) = |F_\Delta|$ (since $1/24 > 1/36$).

**Pure-X and pure-Y orbits are both lower energy than the diagonal orbit.**

### 3.6 Stability (Hessian in 4D)

At pure-X $(R, 0, 0, 0)$ in $(a, b, c', d)$ coords:
$\partial^2 F/\partial a^2 = \mu + 6\Lambda(3a^2 + b^2 + c'^2 + d^2) + 6\Lambda(c'^2 + d^2) = \mu + 18\Lambda R^2 = -2\mu > 0$. Stable (radial).
$\partial^2 F/\partial b^2 = \mu + 6\Lambda(a^2 + 3b^2 + c'^2 + d^2) + 6\Lambda(c'^2+d^2) = \mu + 6\Lambda R^2 = 0$. **Goldstone (X-phase, translation).**
$\partial^2 F/\partial c'^2 = \mu + 6\Lambda(a^2 + b^2 + 3c'^2 + d^2) + 6\Lambda(a^2+b^2) = \mu + 6\Lambda R^2 + 6\Lambda R^2 = -\mu > 0$. Stable.
$\partial^2 F/\partial d^2 = -\mu > 0$. Stable.
All cross-derivatives vanish at $(R, 0, 0, 0)$.

**Hessian spectrum at pure-X:** $\{-2\mu, 0, -\mu, -\mu\}$ = $\{2|\mu|, 0, |\mu|, |\mu|\}$ (all $\geq 0$; 1 Goldstone).

**Pure-X is a minimum modulo 1 Goldstone direction** (the $b$-direction = X-translation).

**At diagonal $(s, 0, s, 0)$:**
$\partial^2 F/\partial a^2 = \mu + 18\Lambda s^2 + 6\Lambda s^2 = \mu + 24\Lambda s^2$. With $\mu = -18\Lambda s^2$: $= 6\Lambda s^2 > 0$.

Hmm let me recompute. $\partial^2 F/\partial a^2$ at $(s, 0, s, 0)$: from $\partial F/\partial a = a[\mu + 6\Lambda(a^2+b^2+c'^2+d^2) + 6\Lambda(c'^2+d^2)]$... partial w.r.t. $a$:
$= [\mu + 6\Lambda(a^2+b^2+c'^2+d^2) + 6\Lambda(c'^2+d^2)] + a \cdot 12\Lambda a$
$= \mu + 6\Lambda(3a^2 + b^2 + c'^2 + d^2) + 6\Lambda(c'^2 + d^2)$.
At $(s, 0, s, 0)$: $= \mu + 6\Lambda(3s^2 + 0 + s^2 + 0) + 6\Lambda(s^2 + 0) = \mu + 24\Lambda s^2 + 6\Lambda s^2 = \mu + 30\Lambda s^2$.
With $\mu = -18\Lambda s^2$: $= 12\Lambda s^2 > 0$. Stable.

$\partial^2 F/\partial b^2$ at $(s, 0, s, 0)$: similarly $\mu + 6\Lambda(a^2 + 3b^2 + c'^2 + d^2) + 6\Lambda(c'^2 + d^2)$
$= \mu + 6\Lambda(s^2 + 0 + s^2 + 0) + 6\Lambda(s^2 + 0) = \mu + 18\Lambda s^2 = 0$. **Goldstone (X-phase).**

$\partial^2 F/\partial d^2 = 0$ by Y $\leftrightarrow$ X symmetry. **Goldstone (Y-phase).**

$\partial^2 F/\partial c'^2$: swap $a \leftrightarrow c'$, same as $\partial^2 F/\partial a^2$: $= 12\Lambda s^2 > 0$.

**Cross:** $\partial^2 F/\partial a \partial c'$ from $\partial F/\partial a = a\cdot[\ldots]$, $\partial/\partial c'[\ldots] = 12\Lambda c' + 12\Lambda c' = 24\Lambda c'$.
So $\partial^2 F/\partial a \partial c' = 24 \Lambda ac'$. At $(s, 0, s, 0)$: $= 24\Lambda s^2$.

$(a, c')$-block Hessian: $\begin{pmatrix} 12\Lambda s^2 & 24\Lambda s^2 \\ 24\Lambda s^2 & 12\Lambda s^2 \end{pmatrix}$.
Eigenvalues: $12\Lambda s^2 \pm 24\Lambda s^2 = \{36\Lambda s^2, -12\Lambda s^2\}$.

**Hessian spectrum at diagonal:** $\{36\Lambda s^2, -12\Lambda s^2, 0, 0\} = \{(+), (-), (0), (0)\}$.

**Diagonal orbit is a saddle with Morse index 1** (ignoring the 2 Goldstones).

### 3.7 Conclusion for $T^2$

**Orbit types at first pitchfork on 2D torus:**

| Orbit | $F$-value | Hessian signature (tangential) | Goldstone dim | Type |
|---|---|---|---|---|
| Pure-X $(R, 0, 0, 0)$ | $-\mu^2/(24\Lambda)$ | $(+, 0, +, +)$ | 1 | Minimum (modulo X-translation) |
| Pure-Y $(0, 0, R, 0)$ | $-\mu^2/(24\Lambda)$ | $(+, +, +, 0)$ | 1 | Minimum (modulo Y-translation) |
| Diagonal $(s, 0, s, 0)$ | $-\mu^2/(36\Lambda)$ | $(+, 0, -, 0)$ | 2 | Saddle, Morse index 1 |

**$|\mathcal{M}_1(T^2)|$ at $\beta$ just above $\beta_{\mathrm{crit}}^{(2)}$:**
- In continuum limit: $\mathcal{M}_1 \cong (S^1 \sqcup S^1)/D_4$. The two circles are exchanged by the $D_4$ action (90° rotation $x \leftrightarrow y$), giving a single circle after quotient.
- $|\mathcal{M}_1| = 1$ (continuum moduli is 1-dim circle).
- At finite $L$: circle is discretized to $L$ standing-wave positions per direction, all equivalent under $D_4 \ltimes (\mathbb Z_L)^2$. So $|\mathcal{M}_1| = 1$ at finite $L$ as well.

**Summary (Round 4 result for $T^2$):**

> **$T^2$ First-Pitchfork Theorem.** On the 2D torus $C_L \times C_L$ ($L \geq 5$) at $c = 1/2$, the reduced Lyapunov function at first Fiedler pitchfork is $F(r_1, r_2) = \tfrac{\mu}{2}(r_1^2 + r_2^2) + \Lambda[\tfrac{3}{2}(r_1^2+r_2^2)^2 + 3r_1^2 r_2^2]$. **Pure-X and pure-Y orbits** are degenerate minima (modulo Goldstone translation direction) with energy $-\mu^2/(24\Lambda)$. **Diagonal orbit** is a saddle (Morse index 1) with higher energy $-\mu^2/(36\Lambda)$. Therefore $|\mathcal{M}_1(T^2)| = 1$ at $\beta$ just above $\beta_{\mathrm{crit}}^{(2)}$, with orbit consisting of **$2L^2$ standing-wave configurations** (vs $D_4$'s 4 on free-BC square grid).

**Category: Cat A** — exact discrete integrals, elementary algebra.

## §4. Comparison: 2D square (free BC) vs 1D cycle vs 2D torus

Unified table of first-pitchfork orbit structure at $c = 1/2$:

| Graph | $\mathrm{Aut}(G)$ | Fiedler dim | $A_2/A_1$ | Isotropy broken? | Orbit selection | $|\mathcal{M}_1|$ | Min orbit size | Goldstones |
|---|---|---|---|---|---|---|---|---|
| 2D square (free BC) | $D_4$ | 2 | **4** | Yes (4 > 2) | **Axis-aligned** (Round 3) | 1 | 4 | 0 |
| 1D cycle $C_n$ | $D_n$ | 2 | **2** | No (2 = 2) | Continuous ring | 1 | $n$ | 1 (in continuum) |
| 2D torus $T^2$ | $D_4 \ltimes (\mathbb Z_L)^2$ | 4 | **(within X/Y): 2; inter-block: 6** | Partially (X-Y cross term breaks isotropy **between** X and Y blocks but not **within**) | **Pure-X or Pure-Y** (not diagonal) | 1 (X and Y linked by $D_4$) | $2L^2$ | 1 per direction; continuous in the orbit |

**Key observations:**

1. **$A_2/A_1 = 2$ is the $O(2)$-invariant point.** It arises whenever the Fiedler subspace is a **free translation-representation** (not a generic $D_4$ 2D rep). The 1D cycle and each block of the torus both have this structure.

2. **2D torus is "two 1D cycles × D_4 gluing".** The block structure (X-modes and Y-modes separately $O(2)$-invariant) is glued by the cross-term $+3 r_1^2 r_2^2$, which breaks the full $O(2) \times O(2)$ to the orbit hierarchy {pure-X or pure-Y} < {diagonal}. The inter-block anisotropy is $A_2^{XY}/A_1 = 4$ (using $A_1 = 3\Lambda/2$ and $A_2^{XY} = 6\Lambda$) — same ratio as 2D grid! This is not a coincidence: the integral $K^{XY} = \int \phi_X^2 \phi_Y^2 = 1$ for both 2D grid and 2D torus (both have product-separable structure in the X-Y direction).

3. **Goldstones reflect continuous symmetry.** 1D cycle has 1 Goldstone per minimum (from $O(2)$ at quartic order, though exact-$D_n$ breaks it at sextic). 2D torus has 1 Goldstone per pure-axis orbit (X-translation for pure-X, Y-translation for pure-Y). 2D square with free BC has 0 Goldstones (D_4 is discrete).

4. **Moduli dimension interpretation.** The $|\mathcal{M}_1| = 1$ in all three cases masks a key difference:
   - 2D square: discrete 4-point orbit (moduli 0-dim).
   - 1D cycle: continuous $S^1/D_n$ orbit (moduli 1-dim).
   - 2D torus: continuous $(S^1 \sqcup S^1)/D_4$ orbit (moduli 1-dim after $D_4$ quotient).

So **"$|\mathcal{M}_1|$ as a set-cardinality" is not the sharp invariant** — the **moduli dimension** is. Continuous-Aut graphs have positive-dimensional moduli; discrete-Aut have 0-dim.

## §5. Category classification and residuals

### 5.1 New Cat A claims (Round 4)

1. **$C_n$ first-pitchfork radial reduction.** $F(r) = \tfrac{\mu}{2}r^2 + \tfrac{3\Lambda}{2}r^4$; 1-dim circle of minima; orbit size $n$; $|\mathcal{M}_1| = 1$.

2. **$T^2$ first-pitchfork orbit selection.** Pure-X or pure-Y orbits are minima (linked by $D_4$); diagonal orbit is saddle; orbit size $2L^2$; $|\mathcal{M}_1| = 1$.

3. **Ratio $A_2/A_1$ classification.** The 2D grid value $A_2/A_1 = 4$ is characteristic of **product-structure $D_4$**; the cycle value $A_2/A_1 = 2$ is characteristic of **single-direction translation** (isotropic / $O(2)$-invariant). These two values are universal across all product-structure and single-cycle graph classes.
