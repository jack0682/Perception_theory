## Style discipline

All figures (data and schematic) follow the same **academic-infographic** constraints:

- Strictly monochrome (black on white, no color, no gradients, no decoration).
- Sans-serif typography, thin line weights (0.6 pt default).
- Rounded rectangles only where structurally meaningful.
- No people, objects, photorealism, hand-drawn aesthetic.
- No text labels in math mode (Greek letters spelled out: `lambda`, `mu`, `omega`, etc.) when generated by image-AI; LaTeX math is allowed in matplotlib via standard mathtext.

See `_style.py` for the shared matplotlib `rcParams`. For schematic figures, see the STYLE GUARD block previously distributed for image-AI prompts.
