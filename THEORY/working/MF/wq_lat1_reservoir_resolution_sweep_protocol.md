## 2. Motivation from Latent Index Space Design

The latent index space design memo (`latent_index_space_design.md` §16) proposes the reservoir reinterpretation: $K_{\mathrm{field}}$ is a finite-rank chart on a latent reservoir $\mathcal A$, and the field-native observable $K_{\mathrm{soft}}^\phi(U)$ should be reservoir-level invariant up to discretization error.

**Three converging predictions** (from §10 and §16 of the design memo):

1. As $K_{\mathrm{field}}$ increases past the reservoir's effective rank $K^*$, the aggregate field $U_K = \sum_j u^{(j)}$ should stabilize.
2. The persistence-based morphology counts $K_{\mathrm{bar}}^{\ell_{\min}}(U_K)$ and $K_{\mathrm{soft}}^\phi(U_K)$ should stabilize.
3. The labelled chart-level count $K_{\mathrm{act}}^\varepsilon$ may continue to depend on $K_{\mathrm{field}}$ — that dependence is *expected* under the reservoir reading because $K_{\mathrm{act}}$ is chart-dependent.

WQ-LAT-1 tests (1) and (2) directly; (3) is documented as a separability check (LAT-C6).

## 3. Formal Reservoir Resolution Hypothesis

**WQ-LAT-1 Reservoir Resolution Hypothesis.** Let $\mathbf u_K(t) \in \widetilde\Sigma^K_M(G)$ be trajectories generated by a fixed SCC multi-field protocol with increasing $K = K_{\mathrm{field}}$. Let

$$
U_K(t) := \sum_{j=1}^{K} u_K^{(j)}(t).
$$

If finite $K_{\mathrm{field}}$ is a finite-resolution chart on a latent formation reservoir, then there exists an effective rank $K^*$ such that for all $K, K' \ge K^*$, the field-native aggregate observables

$$
K_{\mathrm{soft}}^\phi(U_K(T)), \qquad K_{\mathrm{bar}}^{\ell_{\min}}(U_K(T)), \qquad \mathrm{Dgm}_0^{\sup}(U_K(T))
$$

agree up to controlled tolerances $(\tau_{\mathrm{soft}}, \tau_{\mathrm{bar}}, \tau_{\mathrm{Dgm}})$ for all $\ell_{\min}$ in the test sweep.

**Non-conclusion.** If no such stabilization is observed across the tested range, the current finite-slot Layer II implementation does *not* behave as a convergent reservoir chart under the chosen dynamics. This refutes neither the reservoir framework as a whole nor SCC; it indicates that *under this dynamics + this initialization*, the reservoir reading does not produce convergence, and other dynamics (Option D-1 joint projection, intervention) or other initialization options (LAT-B dust reservoir, LAT-C split-bump refinement) should be considered.

## 5. Experimental Design

### 5.1 Baseline configuration

Match WQ-1 / WQ-2 / WQ-1.C-R2 baseline as closely as possible:

| Parameter | Value |
|---|---|
| Graph $G$ | $T^2_{20}$ (4-regular periodic, $|X| = 400$) |
| Total mass $M$ | 90 |
| Per-cluster mass | 30 (always 3 clusters; sub-atom mass = 30 / cluster size) |
| Activity threshold $\varepsilon$ | 0.225 ($= 0.01 \cdot 30 = 0.01 \cdot$ cluster mass) |
| Persistence threshold sweep $\ell_{\min}$ | $\{0.05, 0.10, 0.15, 0.20\}$ |
| Soft-count envelope $\phi$ | $\phi(\ell) = \ell / (\ell + \ell_{\min})$ at $\ell_{\min} = 0.10$ (φ-sat family) |
| Bump width $\sigma_b$ | 2.0 |
| Inter-formation repulsion $\lambda_{\mathrm{rep}}$ | 10.0 |
| Simplex barrier $\lambda_{\mathrm{bar}}$ | 100.0 |
| Time step $dt$ | $10^{-3}$ initial; adaptive |
| Max iter | 5000 |
| Snapshot interval | 25 iter |
| Seed | 42 |

### 5.2 K sweep

$K_{\mathrm{field}} \in \{3, 4, 6, 8, 12\}$. Five trajectories per (option, geometry) pair.

### 5.3 Geometries

- **A — equilateral:** base 3-cluster centers $\{(5, 5), (15, 5), (10, 14)\}$ (matching WQ-1 A).
- **B — compressed isosceles:** base 3-cluster centers $\{(5, 5), (15, 5), (10, 11)\}$ (matching WQ-1 B).

### 5.4 Output volume

5 $K$ × 2 options × 2 geometries = **20 trajectories** maximum. Each ~5–10 s wall-clock for 5000 iter on $T^2_{20}$. Total compute ~3 minutes.

## 6. Initialization Options

The five-$K$ sweep would be straightforward if all $K$ values had a unique correct initialization. They do not. Three options are catalogued; LAT-A and LAT-C are run; LAT-B is reserved for follow-up.

### 6.1 LAT-A — Inactive extra slots

For $K \ge 3$:

- Active slots $j = 1, 2, 3$ initialized at the 3 base centers with mass 30 each.
- Slots $j = 4, \ldots, K$ initialized to zero with mass 0.
- The `active_slot_mask` of `nq242c_counterexample.run_trajectory` skips zero-mass slots from gradient updates and from the Option D-2 rescale, so they remain at zero indefinitely.

**Advantages.** Closest to WQ-1 / WQ-2 baseline. Tests whether merely *having more architectural cap* changes the dynamics.

**Disadvantages.** Inactive slots are *frozen*: they do not exchange mass with active slots. Under this option the dynamics for $K = 3, 4, 6, 8, 12$ is identical (only architectural cap changes). The aggregate field $U_K(t)$ is identical across $K$ values within numerical precision; convergence is *trivial*. **LAT-A serves as a control / null test.** A non-trivial reservoir-resolution test must use LAT-C.

### 6.2 LAT-B — Small dust reservoir (reserved for follow-up)

For $K > 3$, initialize extra slots with a small-amplitude diffuse field (e.g. $u^{(j)}(0) = 0.01 \cdot \mathbb 1$ uniformly, with very small total mass $\le \varepsilon$ per slot).

**Advantages.** Tests whether latent-but-near-threshold slots can absorb mass from the dynamics, supporting a "reservoir activation" reading.

**Disadvantages.** Requires careful $\varepsilon$ tuning. If dust slots are above $\varepsilon$, $K_{\mathrm{act}}$ is artificially inflated. If below, the simplex barrier and Option D-2 rescale may push them to zero quickly.

**Status.** Not run in WQ-LAT-1 first iteration. Documented as future option if LAT-A / LAT-C results are inconclusive.

### 6.3 LAT-C — Split-bump refinement

For $K \ge 3$, distribute $K$ atoms across the 3 base clusters with sub-bump centers in a small ring around each cluster center.

**Atom distribution per $K$:**

| $K$ | atoms per cluster | per-atom mass |
|---|---|---|
| 3 | $(1, 1, 1)$ | $(30, 30, 30)$ |
| 4 | $(2, 1, 1)$ | $(15, 15, 30, 30)$ |
| 6 | $(2, 2, 2)$ | $(15, 15, 15, 15, 15, 15)$ |
| 8 | $(3, 3, 2)$ | $(10, 10, 10, 10, 10, 10, 15, 15)$ |
| 12 | $(4, 4, 4)$ | $(7.5)_{12}$ |

For each cluster of $n \ge 2$ sub-atoms, place sub-atoms in a ring of radius $r_{\mathrm{ring}} = 1.5$ around the cluster center, equally spaced in angle. For $n = 1$, place at the cluster center itself.

**Advantages.** All slots are active throughout the trajectory. Each slot carries non-trivial mass. The dynamics genuinely depends on $K$. The aggregate $U_K$ is the sum of $K$ overlapping sub-bumps that, taken together, reconstruct the original 3-cluster pattern at $K = 3$. As $K$ increases, the sub-bumps refine the spatial resolution of the reservoir.

**Disadvantages.** Within each cluster, sub-atoms heavily overlap (separation ~3, σ_b = 2.0), so the simplex barrier may activate aggressively at $t = 0$. The build_initial_state simplex-enforcement loop handles this by uniform scaling. Per-atom mass at $K = 12$ is small (7.5), so per-atom max amplitude is small; saturation is minimal.

**Use as primary reservoir-resolution test.**

### 6.4 Recommendation

- Run **LAT-A** as control / null test. Expected: trivial agreement across $K$ (because dynamics is identical).
- Run **LAT-C** as primary test. Expected: non-trivial dynamics; observable convergence behavior tests the reservoir hypothesis.
- LAT-B reserved for follow-up if LAT-A or LAT-C is inconclusive.

## 7. Dynamics

### 7.1 Energy

The Layer II energy from `nq242c_counterexample.py` (canonical SCC 4-term per slot + bilinear inter-slot repulsion + simplex barrier):

$$
\mathcal E_K(\mathbf u) = \sum_{j=1}^{K} \mathcal E_{\mathrm{self}}(u^{(j)}) + \lambda_{\mathrm{rep}} \sum_{j < k} \langle u^{(j)}, u^{(k)} \rangle + \lambda_{\mathrm{bar}} \sum_x \max(0, \textstyle\sum_j u^{(j)}(x) - 1)^2.
$$

### 7.2 Integrator

Option D-2 (per-slot gradient + uniform total-mass rescale, with `active_slot_mask`), exactly as in `nq242c_counterexample.run_trajectory`. The script reuses `run_trajectory` directly.

### 7.3 Active slot mask

- LAT-A: slots with $m_j(0) = 0$ are masked from gradient updates and from the D-2 rescale. They remain at zero throughout.
- LAT-C: all slots are active ($m_j(0) > 0$ for all $j$).

This is the WQ-1 / WQ-2 / WQ-1.C-R2 mask convention — preserves baseline dynamics.

## 8. Diagnostics

For every (K, option, geometry) trajectory, at every snapshot $t = \tau_i$:

| Diagnostic | Source |
|---|---|
| Per-slot mass $m_j(t)$ | direct sum |
| $K_{\mathrm{act}}^\varepsilon(t) = \#\{j : m_j > \varepsilon\}$ | direct |
| Aggregate field $U_K(t) = \sum_j u^{(j)}(t)$ | sum over slots |
| $H_0$ superlevel barcode of $U_K(t)$ | `ksoft_kact_diagnostics.h0_barcode` |
| $K_{\mathrm{bar}}^{\ell_{\min}}(U_K(t))$ for $\ell_{\min} \in \{0.05, 0.10, 0.15, 0.20\}$ | sort barcode + count |
| $K_{\mathrm{soft}}^\phi(U_K(t))$ at default $\ell_{\min} = 0.10$ | `ksoft_kact_diagnostics.k_soft_phi_sat` |
| dominant bar lengths (top 8) | sort barcode |
| $\mathcal F(U_K(t))$ — strict local maxima | `ksoft_kact_diagnostics.f_count` |
| pairwise overlaps $O_{jk}(t) = \langle u^{(j)}, u^{(k)} \rangle$ (matrix $K \times K$) | dot products |

For cross-$K$ analysis (final snapshot only):

| Diagnostic | Source |
|---|---|
| Aggregate-field final-state distance: $\|U_K(T) - U_{K'}(T)\|_2 / \sqrt{|X|}$ | $L^2$ |
| Aggregate-field final-state distance: $\|U_K(T) - U_{K'}(T)\|_\infty$ | $L^\infty$ |
| Dominant-bar-vector distance: $\|\boldsymbol\ell_K - \boldsymbol\ell_{K'}\|_2$ | top 6 bars |
| $K_{\mathrm{bar}}^{\ell_{\min}}$ agreement: $K_{\mathrm{bar}}(U_K) - K_{\mathrm{bar}}(U_{K'})$ at each $\ell_{\min}$ | direct integer diff |
| $K_{\mathrm{soft}}^\phi$ difference: $\|K_{\mathrm{soft}}^\phi(U_K) - K_{\mathrm{soft}}^\phi(U_{K'})\|$ | scalar |

σ extraction via `compute_sigma_rich(U_K(T), ...)` is **optional** (--with_sigma flag). Default off because FD-Hessian is $O(|X|^2)$ per snapshot; running it on 20 snapshots adds ~10 minutes of compute. WQ-LAT-1 does not require σ for the primary criteria.

## 9. Convergence Criteria LAT-C1–LAT-C7

### LAT-C1 — Protocol consistency

All $K$ values use the same graph, mass, energy parameters, integrator, time step, max iter, snapshot interval, $\ell_{\min}$ sweep, seed (within an option). Only $K_{\mathrm{field}}$ and the initialization vector vary.

**Status criterion:** PASS by construction (single shared `RunConfig` template per option).

### LAT-C2 — Aggregate final-state convergence

There exists $K^* \in \{3, 4, 6, 8, 12\}$ such that, for all $K, K' \ge K^*$,

$$
\|U_K(T) - U_{K'}(T)\|_2 / \sqrt{|X|} \le \tau_U,
$$

with $\tau_U = 0.05$ default (relative tolerance on per-vertex aggregate value).

### LAT-C3 — $K_{\mathrm{bar}}$ trajectory stabilization

For all $K, K' \ge K^*$ and all $\ell_{\min}$ in the sweep:

$$
K_{\mathrm{bar}}^{\ell_{\min}}(U_K(T)) = K_{\mathrm{bar}}^{\ell_{\min}}(U_{K'}(T)).
$$

(Integer-valued; equality required.)

### LAT-C4 — $K_{\mathrm{soft}}^\phi$ convergence

For all $K, K' \ge K^*$:

$$
\big| K_{\mathrm{soft}}^\phi(U_K(T)) - K_{\mathrm{soft}}^\phi(U_{K'}(T)) \big| \le \tau_{\mathrm{soft}},
$$

with $\tau_{\mathrm{soft}} = 0.10$ default.

### LAT-C5 — Persistence profile convergence

Top-6 dominant bar lengths $\boldsymbol\ell_K = (\ell_K^{(1)}, \ldots, \ell_K^{(6)})$ stabilize: for all $K, K' \ge K^*$,

$$
\| \boldsymbol\ell_K - \boldsymbol\ell_{K'} \|_2 \le \tau_{\mathrm{Dgm}},
$$

with $\tau_{\mathrm{Dgm}} = 0.10$ default.

### LAT-C6 — $K_{\mathrm{act}}$ / $K_{\mathrm{soft}}$ separation documented

Record whether $K_{\mathrm{act}}^\varepsilon(\mathbf u_K(T))$ varies with $K$ (expected under reservoir reading) while $K_{\mathrm{soft}}^\phi(U_K(T))$ remains stable. The reservoir framework predicts $K_{\mathrm{act}}$ is chart-dependent and $K_{\mathrm{soft}}$ is field-native; a *separation* (different convergence behaviors) is the clearest empirical signature.

**Status criterion:** PASS iff (LAT-C4 holds) AND (range of $K_{\mathrm{act}}$ across $K$ exceeds the range of $K_{\mathrm{soft}}$ in normalized units).

### LAT-C7 — A/B geometry consistency

If both A and B are run, both should exhibit qualitatively similar convergence behavior (both converge or both fail). If A converges and B does not (or vice versa), record that geometric initial state affects convergence.

## 10. Failure Modes LAT-F1–LAT-F7

| Mode | Condition |
|---|---|
| LAT-F1 — extra slots inactive | LAT-A only: dynamics identical across $K$, trivial agreement |
| LAT-F2 — artificial mass injection | projection injects mass into nominally inactive slots |
| LAT-F3 — no convergence | $K_{\mathrm{soft}}$, $K_{\mathrm{bar}}$, persistence profiles vary with $K$ |
| LAT-F4 — option dependence | LAT-A and LAT-C disagree |
| LAT-F5 — threshold artifact | convergence only at one $\ell_{\min}$ |
| LAT-F6 — dynamics instability | numerical instability for larger $K$ |
| LAT-F7 — runtime infeasible | sweep cannot complete |

LAT-F1 is **expected for LAT-A** (it is a control test by design). LAT-F1 is a *failure mode for LAT-C* if the dynamics happens to lock all sub-atoms in place.

## 13. Interpretation Rules

### Success (LAT-C1..C5 PASS, LAT-C6 documented, LAT-C7 holds across A and B)

> The tested protocol supports the reservoir interpretation: field-native aggregate observables ($U$, $K_{\mathrm{bar}}$, $K_{\mathrm{soft}}$, dominant-bar profile) stabilize beyond an effective $K^*$. Finite active-slot bookkeeping ($K_{\mathrm{act}}$) remains chart-dependent. The empirical record is consistent with $K_{\mathrm{field}}$ being a finite-resolution chart on a latent reservoir.

### Weak success

LAT-C3 + LAT-C4 PASS but LAT-C2 fails: morphology-level convergence without exact field convergence. Acceptable evidence; the reservoir's *topological* observables converge while the *spatial* aggregate retains $K$-dependence.

### Failure (no convergence)

> The current finite-slot implementation does not behave as a convergent reservoir chart under this dynamics. The reservoir framework remains a candidate at the conceptual level, but the measure-theoretic Model I prediction is not realized by Option D-2 dynamics.

### Inconclusive (LAT-A only)

LAT-A trivially passes LAT-C2 / C3 / C4 / C5 because the dynamics is identical across $K$ (extra slots are frozen). This is a control test; it does not test the reservoir-resolution prediction.

LAT-C runs on top of LAT-A is required for a substantive verdict.

### What is *never* concluded

- σ-rich sufficiency.
- OP-0008 resolution.
- OP-0005 resolution.
- Reservoir framework promotion to canonical.
- Application-domain validity.
