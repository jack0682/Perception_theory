### §2.1 Pre-objective claim 재진술

Commitment 1 (canonical line 757):

이 두 commitment는 SCC의 *pre-objective primacy* — "objects are derivative; the cohesion field is primary" — 를 명시적으로 표현한다. 특히 Commitment 1의 *"not a class label, not an instance ID"* 부분은 multi-object의 *labeling*이 primitive level에 있어서는 안 됨을 의미한다.

### §2.2 K-field architecture의 *prima facie* 위반

I9 (canonical line 846):
> *"K-field architecture: K coupled soft fields $u^1_t, \ldots, u^K_t$ on a product manifold $\Sigma^K_M = \Sigma_{m_1} \times \cdots \times \Sigma_{m_K}$ with inter-field repulsion $\lambda_{\text{rep}} \langle u^j, u^k \rangle$ and simplex participation constraint $\sum_k u^k(x) \leq 1$. The K-field architecture (I9) **guarantees K>1 by construction**."*

**Prima facie 위반 진단**:
- K-field architecture는 K개의 *distinct labeled fields* $u^{(1)}, \ldots, u^{(K_{\mathrm{field}})}$를 가정.
- Index $j \in \{1, \ldots, K_{\mathrm{field}}\}$는 *class label* 또는 *instance ID*처럼 작동.
- "K개의 labeled fields"는 *"K개의 already-individuated objects"* 형태처럼 보임.
- "guarantees K>1 by construction"은 multi-object existence를 *ontological*으로 요구하는 듯한 framing.
- 이는 Commitment 1의 *"not a class label, not an instance ID"*와 *prima facie* 충돌.

## §4. Tool A2 Quotient Resolution — 정밀화

### §4.1 Quotient construction recap

`mathematical_scaffolding_4tools.md` §3 정의:

**Definition (Unordered Configuration Space)**. $S_{K_{\mathrm{field}}}$ acts on $\widetilde\Sigma^K_M$ by index permutation:
$$\pi \cdot (u^{(1)}, \ldots, u^{(K_{\mathrm{field}})}) := (u^{(\pi^{-1}(1))}, \ldots, u^{(\pi^{-1}(K_{\mathrm{field}}))}), \quad \pi \in S_{K_{\mathrm{field}}}.$$

The **unordered shared-pool space** is the orbit space:
$$\widetilde{\widetilde\Sigma}^K_M := \widetilde\Sigma^K_M / S_{K_{\mathrm{field}}}.$$

### §4.2 Pre-objective primacy 자동 만족

**Claim 4.2.1**: 양자화 공간 $\widetilde{\widetilde\Sigma}^K_M$에서, configuration $[\mathbf{u}]$ (orbit class)는 *labels-forgotten* state — 즉 *어느 site가 어느 formation에 속하는가*는 결정되지 않음. 단 *모든 active formations의 spatial supports*만 결정.

**Verification**:
- Orbit $[\mathbf{u}] = \{\pi \cdot \mathbf{u} : \pi \in S_{K_{\mathrm{field}}}\}$.
- 모든 ordered representative $(u^{(\pi(1))}, \ldots, u^{(\pi(K_{\mathrm{field}}))})$가 동일한 orbit class를 결정.
- 즉 *"어느 formation이 #1이고 어느 formation이 #2인가"*는 orbit class 결정에 무관 — 결정되는 것은 *"K_act개의 active formations이 각각 어떤 spatial support를 가지는가"*뿐.
- 이는 정확히 Commitment 1의 *"not a class label, not an instance ID"* 요구 — labels와 IDs는 quotient level에서 자동 forgotten.

**Conclusion**: Pre-objective primacy (Commitment 1)는 unordered configuration $[\mathbf{u}] \in \widetilde{\widetilde\Sigma}^K_M$ level에서 *자동 만족*.

### §4.3 Ordered K-field as modeling-layer lift

**Claim 4.3.1**: Ordered K-field $\mathbf{u} = (u^{(1)}, \ldots, u^{(K_{\mathrm{field}})}) \in \widetilde\Sigma^K_M$는 unordered orbit class $[\mathbf{u}] \in \widetilde{\widetilde\Sigma}^K_M$의 *modeling-layer lift* — 즉 분석 편의를 위해 *labels를 부여한 후의 representation*.

**Verification**:
- Quotient projection $\pi: \widetilde\Sigma^K_M \to \widetilde{\widetilde\Sigma}^K_M$, $\mathbf{u} \mapsto [\mathbf{u}]$.
- Lift $\sigma: \widetilde{\widetilde\Sigma}^K_M \to \widetilde\Sigma^K_M$ (section of $\pi$): orbit class $[\mathbf{u}]$에서 *one ordered representative*를 선택. Section은 *non-canonical* — modeler choice (e.g., ordering by mass: $\|u^{(1)}\|_1 \geq \|u^{(2)}\|_1 \geq \ldots$).
- Lift는 *computational scaffold*: ordered representation에서 SCC 4-energy + λ_rep + simplex constraint 등 standard operations 수행 후, 결과를 quotient로 project (S_K-invariant outputs only).

**Computational scaffold function**:
- SCC `scc/multi.py` 등 numerical implementation은 ordered K-field를 사용 (computational efficiency).
- Theoretical statements (T-Persist-K-*, T-σ-multi-A-*, etc.)는 *S_K-invariant* — 즉 ordered representation에서 진술하더라도 quotient level에서 well-defined.
- Cognitive science mapping은 quotient level (unordered = pre-objective)에서 진행; ordered representation은 *internal computation only*.

### §4.4 CN10 one-way ontological flow

**Claim 4.4.1**: CN10 contrastive ontological flow는 다음과 같이 정확하게 기술 가능:
$$u_t \text{ (Commitment 1 primitive)} \to [\mathbf{u}] \in \widetilde{\widetilde\Sigma}^K_M \text{ (pre-objective unordered)} \to \mathbf{u} \in \widetilde\Sigma^K_M \text{ (modeling-layer ordered lift)} \to (K_{\mathrm{field}}, K_{\mathrm{act}}(t)) \text{ (derived diagnostics per Commitment 16)} \to \text{cog-sci downstream comparisons}.$$

각 화살표는 one-way; 역방향은 CN10 forbidden.

**Verification**:
- **First arrow** ($u_t \to [\mathbf{u}]$): single field aggregate $u = \sum_k u^{(k)}$가 multi-mode structure를 emergence시킴. Quotient level에서 mode decomposition은 H₀ persistence (Tool A3 zigzag PH) 또는 spectral cluster decomposition으로 derived. *Reverse forbidden*: $[\mathbf{u}]$가 single $u_t$ primitive를 *대체*할 수 없음 (Commitment 1 보호).
- **Second arrow** ($[\mathbf{u}] \to \mathbf{u}$): ordered lift = labeling choice. *Reverse forbidden*: ordered representation을 fundamental로 취급하면 Commitment 1 위반.
- **Third arrow** ($\mathbf{u} \to (K_{\mathrm{field}}, K_{\mathrm{act}})$): K_field = modeling-layer cap (Commitment 16); K_act = derived integer diagnostic (support-thresholding count). *Reverse forbidden*: K_field 또는 K_act가 u_t를 *replace*할 수 없음.
- **Fourth arrow** ($(K_{\mathrm{field}}, K_{\mathrm{act}}) \to$ cog-sci): downstream comparison only. *Reverse forbidden* (CN10 contrastive vs reductive): cognitive science K (Pylyshyn FINSTs, Treisman groups)이 SCC의 K_field 또는 K_act를 *define*할 수 없음 — only *compare*.

## §5. Cognitive Science Integration

### §5.1 Pylyshyn 1989 FINST theory — multi-object tracking

**Reference**: Pylyshyn, Z. W. (1989). "The role of location indexes in spatial perception: A sketch of the FINST spatial-index model." *Cognition*, 32(1), 65-97.

**Core claim**: 시각 시스템은 *4±1*개의 *visual indexes* (FINSTs, "Fingers of Instantiation")로 simultaneously refer to objects in the visual scene. Each FINST가 하나의 object location을 track. K = 4±1는 *attentional system 외부*에서 결정되는 *capacity bound* — visual scene 내부의 ontological feature가 아님.

**SCC mapping (CN10 contrastive)**:
- Pylyshyn K (FINST capacity = 4±1) ↔ SCC K_field (architectural cap, modeling-layer commitment).
- Pylyshyn FINST의 *external nature* (attention system이 K를 set; visual scene 내부에 K가 *내재*하지 않음) ↔ SCC K_field의 *modeling-layer status* (modeler / experimental setup이 K_field를 set; pre-objective $u_t$ field 내부에 K_field가 *내재*하지 않음).
- *Reductive identification forbidden*: SCC K_field가 *"FINST capacity = 4의 mathematical analog"*라고 직접 identify할 수 없음. SCC는 self-contained mathematical theory; cognitive science는 *downstream comparison*.

### §5.2 Treisman 1982 Feature Integration Theory — gestalt binding

**Reference**: Treisman, A. (1982). "Perceptual grouping and attention in visual search for features and for objects." *Journal of Experimental Psychology: Human Perception and Performance*, 8(2), 194-214.

**Core claim**: 시각 시스템은 *features* (color, orientation, motion 등)를 *attentional binding* 통해 *perceptual groups*로 결합. K = perceptual group count는 *attentional binding mechanism*에 의해 결정 — features 자체에 *내재*하지 않음.

### §5.3 외부 cognitive system이 K를 결정한다는 것의 의미

**Both Pylyshyn (FINSTs, K_attention) and Treisman (gestalt groups, K_binding) place K *externally***:
- K가 visual scene / sensory input의 ontological feature가 아님.
- K가 cognitive system의 *attentional / binding mechanism*에 의해 결정.
- K가 scene 내부의 *objects의 수*가 아니라, *cognitive system이 동시에 track / bind할 수 있는 attentional units의 수*.

**SCC의 pre-objective claim과의 reconciliation**:
