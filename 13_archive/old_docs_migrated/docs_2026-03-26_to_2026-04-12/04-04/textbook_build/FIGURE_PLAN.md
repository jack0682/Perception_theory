## P1: 기초 개념 시각화 (우선 작성)

### 1.1 객체 중심 vs. 응집도 중심 (Ch1)
- **유형:** 개념도 (2개 병렬)
- **내용:**
  - 좌: 객체-중심 (이산 경계)
  - 우: 응집도-중심 (연속 필드)
- **도구:** matplotlib / TikZ
- **예제:** 5×5 격자 with Gaussian bump
- **페이지:** Ch1-2.5

### 1.2 Soft Cohesion Field 예제 (Ch2)
- **유형:** 열맵 (heatmap)
- **내용:** u_t : [0,1]^X 필드 시각화
- **사이즈:** 5×5, 10×10 격자
- **색상:** 파란색 (저) → 빨간색 (고)
- **도구:** matplotlib
- **페이지:** Ch2-1.5

### 1.3 Proto-cohesion 벡터의 4개 요소 (Ch2)
**각 요소마다 1-2개 그림:**

**1.3a Bind (응집도)**
- 필드 u_t 시각화
- ℓ²/√n 계산 과정
- "응집도 낮음" vs "높음" 예제 (2개)

**1.3b Sep (분리도)**
- u_t와 (1-u_t) 비교
- 가중 평균 계산 시각화
- "분리 낮음" vs "높음" (2개)

**1.3c Inside (내부성)**
- H_0 동질화를 통한 "핵심" 검출
- Articulation 표시
- "내부성 낮음" vs "높음" (2개)

**1.3d Persist (지속성)**
- 시간 t → t+1 필드 변화
- 겹침 영역 강조
- "지속성 낮음" vs "높음" (2개)

**소계:** 8개 그림

## P2: 형식 이론 다이어그램

- **유형:** 스키마
- **내용:** 
  ```
  Formal Universe C^soft = (T, {X_t}, {u_t}, {Cl_t}, {N_t, D_t}, {M_t->s})
  ```
- **표현:** 박스다이어그램 (각 요소의 역할 표시)
- **도구:** TikZ

### 2.2 폐포 연산자 (Ch4)
- **유형:** 그래프
- **내용:**

  - 폐포 Cl_t(u) 적용 과정

### 2.3 구분 연산자 (Ch4)

- **내용:**

- **도구:** matplotlib
- **페이지:** Ch4-1.5

### 2.4 에너지 항 (Ch5)
