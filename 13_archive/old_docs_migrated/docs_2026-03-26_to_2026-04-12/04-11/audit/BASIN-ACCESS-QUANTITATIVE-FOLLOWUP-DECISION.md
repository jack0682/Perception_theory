## 1. Options

### Option A — Basin-Size Proxy Scaling

Goal:

> measure return probability as a function of perturbation radius more finely, and compare it against raw-start discovery probability.

Pros:
- directly quantifies the “hard to enter, easy to keep” picture;
- extends Exp80 naturally;
- likely gives the cleanest next bridge from qualitative mechanism to quantitative statement.

### Option B — Access-Path Diagnostics

Goal:

> inspect the route by which seeded runs enter the low-energy basin and compare it to raw runs.

Pros:
- more mechanistic;
- potentially closer to a later analytic explanation.

Cons:
- instrumentation is more involved;
- more interpretation-heavy than a clean quantitative scaling curve.

---

## 2. Decision

**Choose Option A first: basin-size proxy scaling.**

Reason:

1. it gives the cleanest quantitative continuation of Exp79 + Exp80;
2. it can be expressed directly as a curve or table of return rates versus perturbation radius;
3. once the scaling is mapped, access-path diagnostics can be targeted more intelligently.

## 3. Next Trigger

Extend Exp80 from a few discrete sigma values to a denser perturbation-radius ladder and compare the resulting return profile against the raw-start access baseline.
