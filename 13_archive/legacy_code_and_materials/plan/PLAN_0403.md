- Neumann series: D^{-1/2}(u)W_sym D^{1/2}(u) 단조성
- 문제: 이 유사성 변환이 u_t(x)에 의존 → 미분에서 단조성 보존 증명 미완

**접근법**:
```
Step 1: D(u)의 u-Jacobian 명시적 계산
        ∂_u D_{ij}(u) = ... (sparse structure 활용)

Step 2: W_sym = D^{-1/2} W D^{-1/2}의 u-미분
        d/du W_sym = f(D, ∂D, W, ∂W) 형태로 정리

Step 3: ||·||_2 norm에서 단조성 보존
        λ_max(∂_u W_sym) ≤ c·λ_max(∂_u W) 형태 부등식

산출물: docs/04-03/proof/C3-SYMMETRIZATION-COMPLETE.md
```

**효과**: C-Axioms 100% 폐쇄 → 정적 이론 완성
