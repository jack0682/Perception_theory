# 다중 형성(Multi-Formation)이 왜 이렇게 어려운가

## 개요

Soft Cognitive Cohesion 이론은 **K=1 (단일 형성)**에 대해서는 완벽하다. 
- 35개의 증명된 정리
- 실험적 검증 완료
- 에너지 함수의 엄밀한 정의

그러나 **K>1 (다중 형성)**로 나아가는 순간, 이론이 무너지기 시작한다.

이 문서는 **왜** 이렇게 되는지, **정확히 어느 지점에서** 문제가 발생하는지를 분석한다.

---

## I. 문제의 본질: "에너지의 말"과 "역학의 행동"의 불일치

### 1.1 에너지가 말하는 것

SCC 이론에서 에너지 함수는 명확하게 하나의 진실을 말한다:

```
K=1 (m=60):    E = 2.25
K=2 (m₁=30, m₂=30): E = 4.66

∴ K=1이 K=2의 약 2배 저렴
∴ 에너지 최소화 원칙에 따르면 K=1이 선택되어야 함
```

이것은 단순한 계산이 아니라, 이론의 **중앙 공리**에서 나온다:

- **A2 공리**: 에너지 최소화 원칙
- **§7 정리들**: E_cl, E_sep, E_bd의 구체적 형태
- **최적화 이론**: Gradient flow는 E의 극소점으로 수렴

따라서 이론의 언어로 하면:
> **"K=2는 에너지 최소화 관점에서 불안정하다. 시스템은 반드시 K=1로 수렴해야 한다."**

### 1.2 그런데 현실에서 우리가 보는 것

그런데 생물학적 세계와 우리의 실험에서는:

- **신경계**: 여러 neuronal assembly가 동시에 활성 (K>1)
- **지각**: 여러 객체를 동시에 인식 (K>1)
- **exp55**: K=2 형성이 5000번 반복에도 merge되지 않음
  - 장벽 높이 ≈20
  - 노이즈 크기 ≈0.5
  - → 장벽이 노이즈의 40배 높음

따라서 현실의 언어로 하면:
> **"K=2는 에너지상 불리함에도 불구하고, 동역학적으로는 매우 안정적이다."**

### 1.3 불일치의 의미

이것은 단순한 "모순"이 아니다. **이론이 설명할 수 없는 현상**의 존재를 의미한다:

```
┌────────────────────────────────────────────────┐
│ 에너지 E의 언어:                              │
│ "K=2는 불가능하다 (K=1이 전역 최소)"         │
├────────────────────────────────────────────────┤
│ 역학의 언어:                                  │
│ "K=2는 매우 오래 지속된다 (높은 장벽)"        │
├────────────────────────────────────────────────┤
│ 우리의 질문:                                  │
│ "둘이 모두 맞는데, 어떻게 양립 가능한가?"    │
└────────────────────────────────────────────────┘
```

**답**: 에너지만으로는 충분하지 않다. **운동학(kinetics)**이 필요하다.

## II. 왜 "K"를 정의하기가 이렇게 어려운가?

### 2.1 K가 명확하지 않은 이유

"K"는 얼핏 보면 간단한 개념이다: "형성의 개수"

하지만 응집장 u_t : X_t → [0,1]의 세계에서 "개수"를 센다는 것은 무엇인가?

#### 문제 1: 경계의 모호성

```
u의 값에 따른 해석:

u(x) = 0.95  → 명확하게 "내부" (형성 1에 속함)
u(x) = 0.50  → ???
u(x) = 0.05  → 명확하게 "외부" (어느 형성에도 속하지 않음)

질문: 0.50은 어느 형성에 속하는가?
     두 형성의 "경계"에 있는가?
     "반쪽씩" 속하는가?
     아니면 이 사이트 자체가 형성의 일부가 아닌가?
```

**문제**: 형성의 정의가 "threshold"에 의존한다.

```
통상적 정의:
K = #{x : u(x) > θ}  (θ = 어떤 threshold)

하지만:
- θ = 0.5를 선택하는 것은 자의적
- θ = 0.8이면 K가 더 커질 수 있음
- θ = 0.9이면 K가 또 달라질 수 있음

결론: K는 객관적 성질이 아니라 "측정 방식"에 의존하는 인공물일 가능성
```

#### 문제 2: 공간적 연결성

```
또 다른 가능한 정의:

K = (connected components의 수)  [u > θ인 영역들]

예시:
u(x) = [0.8, 0.8, 0.1, 0.1, 0.8, 0.8]  (1D array)

만약 θ = 0.5:
Connected components: [0.8,0.8], [0.8,0.8]
→ K = 2

만약 두 bump 사이의 값이 0.4로 올라가면:
Connected components: [0.8,0.8,0.4,0.8,0.8]  (all connected)
→ K = 1

문제: 임의로 작은 값 변화로 K가 "jump"
      이것은 K가 연속 변수(u)로부터 불연속적으로 튀어나온다는 뜻
      이론적으로 매우 불안정
```

#### 문제 3: 다른 정의들의 가능성

```
K를 정의하는 방법들:

방법 A: "局部 극소값의 개수"
   → E(u)의 이중 도함수로 판정
   → 하지만 이것은 에너지 관점일 뿐, "형성"의 개념이 아님

방법 B: "독립적인 응집 영역"
   → "독립"의 정의가 필요 (상호정보량? Hamming distance?)
   → 임의적이고 수학적 근거 약함

방법 C: "응집 공간의 spectral dimension"
   → K를 u의 고유벡터로 정의?
   → 이건 이론의 프레임에서 온 게 아님, 후처리(post-hoc)

방법 D: "에너지 landscape의 basin의 수"
   → 가장 "순수" 한 정의
   → 하지만 여전히 "basin"의 경계 정의 필요
```

**핵심 발견**: K를 정의하는 모든 방법이 **자의적인 선택**을 포함한다.

### 2.2 K의 정의 부재가 의미하는 것

이것이 왜 심각한가?

```
우리 이론의 중앙 명제:
───────────────────
"형성 u_t는 4개 조건을 만족할 때 객체가 된다"
(Bind, Sep, Artic, Persist)

그런데 여기서 빠진 것:
──────────────────
"여러 형성이 동시에 존재할 때는?"

이에 대답하려면:
1. 각 형성을 정의해야 함
2. 하지만 K가 명확하지 않음
3. 따라서 "형성이 몇 개인지" 세어도 의미가 불명확

결론: 우리 이론은 K=1에서만 작동한다.
     K>1은 이론이 아직 "말할 수 없는" 영역.
```

## III. 왜 "K 선택"을 설명할 수 없는가?

### 3.1 K 선택 메커니즘의 부재

우리 시스템에서 K가 결정되는 방식을 생각해보자:

#### 초기 조건에 따른 K 결정

```
실험 관찰:
────────
같은 파라미터(λ, β, etc.)에서도:
• 초기 조건 A → K=1 결과
• 초기 조건 B → K=2 결과
• 초기 조건 C → K=3 결과

이론적 설명?
───────────
에너지 함수 E(u)는 K와 무관하게 정의됨.

따라서:
- K=1이 전역 최소 (항상)
- K=2, K=3는 모두 E가 높음 (K=1보다)

그렇다면:
• 어느 초기 조건이 K=2를 낳는가?
• 왜 일부 초기 조건은 K=1로 수렴하고 다른 것은 K=2로 가는가?
• 이것을 예측할 수 있는가?

현재 이론: "예측 불가"
```

#### 파라미터 변화에 따른 K 변화

```
생물학적 현상:
──────────
• λ_cl 증가 → K=1 → K=2 (bifurcation)
• λ_rep 증가 → K=1 → K=2
• 노이즈 감소 → K의 fluctuation 감소

이론적 설명?
──────────
E(u)는 λ_cl, λ_rep에 종속이지만:
• 모든 λ에 대해 K=1이 여전히 전역 최소
• K=2의 상태공간상 "위치"가 변하지 않음
• "bifurcation" 개념이 없음

결론: 파라미터 변화에 따른 K의 전환을 설명할 수 없다.
     "언제 K=1→K=2인가?"를 예측할 수 없다.
```

### 3.2 K의 동역학이 없다

가장 근본적인 문제는:

```
우리 이론에 있는 것:
──────────────────
du_t/dt = ? (폐포 동역학, 에너지 그래디언트 flow, etc.)

우리 이론에 없는 것:
──────────────────
dK/dt = ?

즉:
• u의 시간 진화는 설명할 수 있다
• 하지만 K가 "어떻게 변하는가"는 설명할 수 없다

더 정확히:
────────
K는 u로부터 "파생되는" 양(derived quantity)이다.
K = f(u)  (어떤 함수 f)

하지만:
• f가 정의되지 않음 (앞의 문제들 때문에)
• 따라서 dK/dt = df(u)/dt를 계산할 수 없음

결론: K는 "observable"이 아니라 "ghost variable"
     있는 것 같지만, 실제로는 이론에 없음.
```

## IV. 지금까지의 시도들: 왜 모두 실패했는가?

### 4.1 시도 1: K-Field 아키텍처 (2026-04-02)

#### 아이디어

```
"K개의 독립적인 응집장 u₁, u₂, ..., u_K를 도입하자"

에너지:
E = Σᵢ(λ_cl·E_cl,i + λ_sep·E_sep,i + λ_bd·E_bd,i) 
    + λ_rep·Σᵢⱼ(i≠j) repulsion(uᵢ, uⱼ)

개념적으로:
• 각 uᵢ는 독립적인 형성
• 반발력 λ_rep이 그들을 분리 유지
• K는 "활성화된" 필드의 개수
```

#### 왜 실패했는가?

```
문제 1: K를 "어디서" 정하는가?
─────────────────────────────
우리는 "K개의 필드"를 가정한다.
하지만:
• K는 외부에서 미리 정해져야 함 (이론 밖에서!)
• "우리 시스템은 K=2 형성을 가져야 한다"고 인간이 정함
• 이것은 이론이 K를 "선택"하는 것이 아니라, 우리가 정하는 것

결론: K는 여전히 "자유도(degree of freedom)"
     이론이 설명하지 않는 외부 파라미터

문제 2: λ_rep는 어떻게 정하는가?
─────────────────────────────────
"반발력을 충분히 크게 해야 두 필드가 겹치지 않는다"

하지만:
• λ_rep의 값이 "얼마나 충분"한지는 문제에 따라 다름
• 일반적인 원칙이 없음
• 각 configuration마다 튜닝 필요 (ad-hoc)

결론: λ_rep는 "이론 밖의 파라미터"
     원리적 근거 없는 튜닝 parameter

문제 3: K>1은 왜 "선호"되는가?
───────────────────────────────
K-field 모델에서:
E_K=1 < E_K=2  (항상)

그렇다면:
• 왜 시스템은 때로 K=2를 "선택"하는가?
• 에너지상으로는 K=1이 더 나은데?
• 초기 조건? 파라미터? 노이즈?

결론: K-field는 "K>1을 만드는 방법"이지,
     "왜 K>1이 선택되는가"를 설명하지 않음.
```

#### 평가

```
K-Field는 이론이 아니라 "구조(scaffolding)"일 뿐이다.

마치:
"여러 객체를 다루고 싶으면, 여러 개의 객체를 가정하고,
 그들이 겹치지 않도록 반발력을 두면 된다"

이것은:
• 계산 도구로서는 유용
• 하지만 "형성이 왜 다중화되는가"를 설명하지 않음
• "K의 선택"은 여전히 미스터리
```

### 4.2 시도 2: Branch Selection & Type A/B (2026-04-07)

#### 아이디어

```
"K=2 내에서도 여러 'branch'가 있을 것이다"

Type A: centered, 안정적
       두 bump이 대칭적, 같은 크기

Type B: off-center, 불안정
       두 bump이 비대칭적, 다른 크기, swap-prone

가설: exp62 vs exp63의 차이가 Type A vs Type B일 것
```

#### 왜 실패했는가?

```
exp65 검증 결과:
───────────────
15×15, c=0.5:  Type A     ✓
15×15, c=0.6:  Mixed boundary
20×20, c=0.5:  Type A     ✓
20×20, c=0.6:  Type A     ✓

Type B 관찰: 0/4 configurations

분석:
────
exp62와 exp63의 차이를 재검토:
• exp62: 전역 최적화 (모든 K=2 상태 탐색)
• exp63: 국소 최적화 (특정 K=2 base point에서 perturbation)

→ 두 가지는 "완전히 다른 optimization problem"
→ 문제를 다르게 푼 것이지, "Type A vs B"가 아님

결론:
────
Type A/B 분류는 검증 불가능한 개념이었다.

우리가 관찰한 것:
• exp62 (broad search) → K=1을 찾음 (에너지 더 낮음)
• exp63 (narrow search) → K=2의 극소값을 찾음 (좁은 영역 내에서)

즉:
"K=2가 branch를 가진다"는 사실이 아니라
"다른 방식으로 최적화하면 다른 결과"라는 당연한 사실
```

#### 평가

```
Type A/B는 우리가 "보고 싶은 것"이었지,
실제 현상이 아니었다.

이 시도는 다음을 배웠다:
• "K=2 내부의 구조"도 중요하지만
• "왜 K=2가 K=1보다 선택되는가"는 여전히 미스터리
```

### 4.3 시도 3: Kramers Rate (2026-04-10 ~ 현재)

#### 아이디어

```
"K=2에서 K=1로의 탈출이 느리다면, 
 K=2는 '동역학적으로' 안정할 수 있다"

Kramers 법칙:
τ ≈ τ₀ exp(B / k_B T)

여기서:
• τ = 탈출 시간 (K=2에서 K=1로 가는 시간)
• B = 장벽 높이
• k_B T = 효과적 온도
• τ₀ = prefactor (시스템의 curvature에 의존)

논리:
────
만약 τ >> t_biological이면:
• 생물학적 timescale에서 K=2는 "안정"
• K=1로 merge될 시간이 없음
• 따라서 K>1이 관찰됨
```

#### 왜 이것이 더 유망한가?

```
이전 시도들과의 차이:
──────────────────

시도 1 (K-field): K를 외부에서 정함
                 → K의 선택 설명 X

시도 2 (Type A/B): K=2 내의 구조만 봄
                  → K 자체의 선택 설명 X

시도 3 (Kramers): K의 "동역학적 안정성"을 봄
                  → K의 선택을 "시간에 따른 dynamics"로 설명

즉:
• 에너지상으로는 K=1 < K=2 (에너지가 말함)
• 하지만 동역학상으로는 K=2에 오래 머무름 (Kramers가 말함)
• 따라서 관찰 timescale에서 K>1이 "유효하게" 안정 (생물학이 말함)
```

#### 하지만 여전한 문제점들

```
문제 1: "반응 좌표(Reaction Coordinate) K"를 정의해야 함
────────────────────────────────────────────────────────
Kramers 법칙의 적용:
τ ≈ τ₀ exp(B / k_B T)

이 공식은 1D potential에서 나온 것:
U(x) = potential along reaction coordinate
B = U(saddle) - U(minimum)

우리의 경우:
• 반응 좌표 = K (형성의 개수)
• 하지만 K는 위에서 본 것처럼 정의가 명확하지 않음!

따라서:
exp81에서 "B를 측정"하려고 하면:
• 장벽 높이 B를 어떻게 측정할 것인가?
• "K=1에서 K=2로 가는 경로"를 어떻게 정의할 것인가?
• 최단 경로? 가장 쉬운 경로? 가장 그럴듯한 경로?

결론: K가 명확하지 않으면 B도 정의 불가능

문제 2: "온도"를 어디서 얻을 것인가?
──────────────────────────────────────
Kramers 공식의 T는 물리적 온도.

우리의 경우:
• 신경계는 물리적 온도가 없음 (또는 약간 있지만 관련 없음)
• 대신 "노이즈" = 불확정한 fluctuation

가정:
"노이즈 amplitude = 효과 온도"

하지만:
• 이 가정이 타당한가?
• 어느 부분의 노이즈? (전체? local? synaptic?)
• 노이즈 분포가 Gaussian인가? (Kramers는 가정)

결론: "온도"의 정의가 ad-hoc

문제 3: Kramers 법칙 자체의 적용 조건
──────────────────────────────────────
Kramers 법칙의 가정:
1. 1D potential (우리: 고차원)
2. 약한 노이즈 (우리: 복잡한 노이즈)
3. Timescale 분리 (빠른 모드 vs 느린 모드)
4. 포텐셜이 smooth (우리: manifold with corners!)

우리 시스템:
• 고차원 응집장 u : X → [0,1]
• 복잡한 상호작용
• 가정 1, 2, 3이 성립하는가? 불명확

결론: Kramers를 "직접" 적용할 수 있는가? 미확실

문제 4: "K=1로의 수렴"이 정말 일어나는가?
─────────────────────────────────────────
우리의 가정:
"K=2는 준정상 상태. 충분히 오래 기다리면 K=1로 수렴."

검증 방법:
exp55에서 5000번 반복 후에도 K=2 유지.

질문:
• 정말 "아직도 수렴 중"인가?
• 아니면 "실제로 K=2가 장기간 안정"인가?

확인 방법:
• 더 오래 run? (그럼 얼마나?)
• exponential convergence 확인? (어떻게?)
• 수치적으로 추정된 τ와 비교? (하지만 τ계산도 불확실)

결론: 근본적 검증의 불가능성
```

## V. 더 깊은 원인: 이론의 구조적 한계

### 5.1 "응집장" 자체로는 K를 설명할 수 없다

핵심 통찰:

```
우리 이론의 최소 원칙:
───────────────────
du_t/dt = (some dynamics based on u_t and energy E(u_t))

이 동역학은:
✓ u → u로의 변환 (필드 내 진화)
✓ 하지만 K는 "세지 않음"

즉:
• u의 dynamics가 주어졌을 때
• K가 어떻게 변하는가는 u의 dynamics로부터 직접 나오지 않음
• K는 u로부터 "파생되는 관찰"일 뿐

다시 말해:
────────
K = f(u)  (어떤 함수 f)

하지만:
• f는 불명확 (앞의 "K 정의" 문제)
• 따라서 K의 dynamics를 계산할 수 없음

결론:
───
"응집장"만으로는 "K"를 설명할 수 없다.
K는 u의 또 다른 "자유도"가 필요하다.
```

### 5.2 "에너지 원리"도 K를 설명할 수 없다

```
에너지 최소화 원칙:
─────────────────
u* = argmin E(u)

이 원칙의 결과:
✓ K=1이 전역 최소 (증명됨, exp51-65로 검증)
✓ 단일 형성의 구조를 완벽하게 설명

하지만:
✗ K>1이 왜 "선택"되는가는 설명 불가

왜냐하면:
────────
E(u)는 K에 대한 "preference"를 갖지 않음.
(K가 정의되지 않았으니까)

더 정확히:
────────
E(u)는 u에 대해서만 정의됨.
E : {u : X → [0,1]} → ℝ

K는 이 함수에 나타나지 않음.
따라서 에너지 원리로부터 K의 선택을 도출할 수 없음.

유일한 방법:
──────────
"K=2 필드"를 이미 가정하고, E를 그 필드에 대해 최소화.
(= K-field 아키텍처)

하지만 이것은:
• K를 "설명"한 것이 아니라
• K를 "가정"한 것
```

### 5.3 "구조적 전환(Phase Transition)"이 없다

```
생물학적 관찰:
──────────────
파라미터 λ를 변화시키면:
• λ < λ_c: K=1이 우세
• λ = λ_c: K=1과 K=2가 경쟁
• λ > λ_c: K=2가 우세

이것은 "phase transition"의 신호.

우리 이론에서:
────────────
파라미터 λ를 변화시켜도:
• 모든 λ에서 K=1이 전역 최소
• "transition point"가 없음

따라서:
────
이론은 "K=1 phase"만 가짐.
"K=2 phase"는 이론의 구조상 존재할 수 없음.

→ 이는 이론이 "K>1을 설명하도록 설계되지 않았다"는 뜻.
```

## VI. 왜 Option C (운동학적 재형성)가 유일한 방법인가?

### 6.1 선택지의 검토

```
방법 1: "K를 버린다"
───────────────────
• K를 쓰지 말고, u의 spectral modes를 직접 분석
• 예: u의 고유벡터 전개
• 장점: 에너지 프레임워크 유지
• 단점: "형성"의 개념을 잃음, 생물학적 해석 불명확

방법 2: "K-field를 받아들인다"
───────────────────────────────
• K는 외부에서 정하는 것으로 받아들임
• K-field 아키텍처만 발전시킴
• 장점: 다중 형성 계산 가능
• 단점: 이론이 아니라 모델링 도구일 뿐, K의 선택 설명 못 함

방법 3: "운동학적 재형성" (Option C)
──────────────────────────────────
• K=2를 "준정상상태"로 재해석
• 에너지: K=1 < K=2 (에너지가 말함)
• 운동학: K=2의 탈출 시간이 길 수 있음 (Kramers가 말함)
• 따라서 "동역학적으로" K=2는 안정할 수 있음
• 장점: 에너지와 현실을 "조화"시킬 수 있음
• 단점: Kramers 적용 가정이 불명확, 검증 어려움
```

### 6.2 운동학적 재형성이 시도할 만한 이유

```
장벽(Barrier)의 존재:
───────────────────
exp55에서 관찰:
• K=2 형성이 5000번 반복에도 유지
• 장벽 높이 ≈ 20 (대략적 추정)
• 노이즈 ≈ 0.5

→ 장벽이 노이즈의 40배

이것은:
• 우연이 아님
• 시스템이 "실제로" K=2에서 머무르고 싶어하는 신호일 가능성
• 에너지상 불리해도 "동역학적으로"는 안정

Kramers 법칙의 적용:
──────────────────
τ ≈ τ₀ exp(B / k_B T)

만약 B ≈ 20, T ≈ 0.5라면:
τ ≈ τ₀ exp(20/0.5) = τ₀ exp(40) ≈ τ₀ × 2.4×10¹⁷

이것은:
• 엄청난 시간 (시스템의 특성 시간의 1017배)
• "생물학적" timescale에서는 무한대에 가까움
• 따라서 K=2는 "유효하게" 안정

→ 이것이 관찰과 부합할 가능성 있음
```

## VIII. 결론: 다중 형성의 어려움의 본질

### 8.1 한 문장 요약

```
다중 형성이 어려운 이유는:
"에너지는 K=1만 선택하는데, 현실은 K>1을 보여주기 때문이다."
```

### 8.2 더 정확한 진단

```
근본적 문제들:

1️⃣ K의 정의 부재
   → K를 명확하게 정의할 수 없음
   → K를 센다는 것 자체가 임의적
   
2️⃣ K 선택 메커니즘 부재
   → 왜 K=2가 선택되는가를 설명할 수 없음
   → K의 dynamics가 없음
   
3️⃣ 에너지만으로는 불충분
   → 에너지 최소화는 K=1만 선택
   → K>1을 에너지로 설명할 수 없음
   
4️⃣ 이론의 "K-blind" 설계
   → 응집장 u의 dynamics는 있음
   → 하지만 K에 대한 dynamics는 없음
   
5️⃣ 수학적 기초의 부실
   → Manifold with corners 미처리
   → Phase transition 개념 부재
```

이 질문에 대답하려면, 먼저 X (즉, "형성"과 "K")를 명확히 정의해야 한다.
