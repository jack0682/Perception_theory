# Canonical Specification v1.2 Release Notes

**Version:** 1.2  
**Release Date:** 2026-04-12  
**Status:** Current (includes audit revisions)  
**Focus:** Transparent acknowledgment of critical open issues

---

## Summary

Canonical Specification v1.2 is a **truthful restatement** of the theory incorporating the comprehensive 2026-04-12 audit findings. Rather than hiding critical vulnerabilities, v1.2 makes them explicit: which theorems depend on unresolved assumptions (F-1, M-1, MO-1), which empirical claims failed (Type A/B classification), and what exact constraints are required for each result.

**Philosophical shift:** Move from implicitly assuming limitations are "known" to *explicitly stating them* in the spec.

---

## Major Changes from v1.1

### 1. Explicit Assumption Statements (New)

**What was added:**
- Every theorem now includes an "Assumptions" section
- "Fixed K" constraint explicitly stated for K-field theorems
- "Fixed per-formation mass" constraint explicitly stated
- "Closed system" (no mass flux) explicitly noted

**Example formulation:**
```
Theorem T-Persist-K-Sep (Well-Separated Regime)
Assumptions: K=2 (externally fixed), m₁=m₂=M/2 (externally fixed),
  formations well-separated (Sep > threshold)
Statement: ...
Category: B (conditional on above assumptions)
```

**Impact:** Readers can now see exactly which theorems are self-contained vs which depend on external scaffolding.

### 2. F-1, M-1, MO-1 Explicit Documentation (New)

**Section added: §14.1 Critical Open Problems**

| Problem | Severity | Status | Impact |
|---------|----------|--------|--------|
| **F-1: K=2 Vacuity** | 🔴 CRITICAL | Unresolved | K=2 stability requires external m constraint; K=1 always cheaper energetically |
| **M-1: K=1 Preference** | 🔴 CRITICAL | Unresolved | M₂ energy landscape monotonically decreasing toward K=1 |
| **MO-1: Morse Failure** | 🟠 HIGH | Unresolved | M₂ is manifold with corners; smooth Morse theory inapplicable |

**Mitigation strategies documented:**
- **F-1 Option A:** Accept "fixed K is external constraint" (current spec approach)
- **F-1 Option B:** Introduce K-selection mechanism (BIC, free energy, birth-death dynamics)
- **F-1 Option C:** Acknowledge K>1 as metastable, not globally optimal
- **MO-1 Option A:** Use existing stable results (no full Morse analysis needed)
- **MO-1 Option B:** Transition to stratified Morse theory (more complex)

**Explicit disclaimer:**
> "Theorems T-Persist-K-Sep, T-Persist-K-Weak, T-Persist-K-Unified depend on resolving F-1 (K determination). Until F-1 is resolved, these theorems hold only under the external assumption that K is fixed. We do not have a mechanism explaining *why* K would be fixed in a biological or cognitive system."

### 3. Type A/B Classification Retracted (Revised)

**What was removed:**
- 04-07 "Type A vs Type B" classification framework
- Claimed divergence between exp62 (Type A) and exp63 (Type B)
- Proposed characterization of K=2 stability types

**Why:**
- exp65 Formation Tracking: 0/4 configurations exhibited Type B behavior
- All configs clustered at Type A or boundary
- Divergence between exp62/exp63 attributed to optimizer strategy, not K-field type
- Classification was unvalidated hypothesis; has no empirical support

**Status:** Framework retracted from canonical; diagnostic data preserved in archive for future analysis.

**Lesson:** Do not introduce unvalidated classification schemes into canonical spec.

### 4. Transport Kernel Form (Refined but Provisional)

**No change to formal statement, but clarification added:**

- Current form: Entropy-regularized optimal transport (Sinkhorn with ε_OT = 1.0)
- **Status:** Provisional; exact form may be refined in later versions
- **Validation:** Confinement bounds 2.4–3.5× (exp45–exp47)
- **Open question:** Is this the unique transport kernel satisfying axioms E1–E5, or one of several realizations?

**Category:** A for transport existence and bounds; C for full morphological inheritance (composition across 3+ time steps).

## Assumptions Made Explicit

### For All Theorems
1. **Relational support fixed:** X_t structure does not change (no dynamic topology)
2. **Closed system:** No mass flux (Σ u_t = const)
3. **Generic parameters:** Results on residual set (Sard regularity)
4. **Finite graph:** |V| = n is finite (no scaling limit)

### For K-field Theorems (T-Persist-K-*, etc.)
1. **Fixed K:** Number of formations K is externally imposed (F-1 issue)
2. **Fixed per-formation mass:** m_j constant (or Σ m_j = M constant) (F-1/M-1 issue)
3. **Volume constraint:** ∫ u^j_t dx = m_j exactly
4. **Closed system:** No K-field merge/birth dynamics (Kramers theory separate)

### For Morse Theory Results
1. **Smooth manifold:** Σ_m is smooth manifold (MO-1 issue: false for Σ²_M)
2. **Generic transversality:** Solutions cross critical surfaces transversely

## Impact on Applications & Future Work

### For Implementers
- Use **single-formation results (T-1:T-14)** with confidence (Category A, no caveats)
- Use **K-field results (T-Persist-K-*, etc.)** with explicit understanding that K is externally fixed
- Do NOT rely on K-field results to explain *how* K emerges in nature

### For Theorists
- **Priority 1:** Resolve F-1 (K selection mechanism) or accept external constraint
- **Priority 2:** Clarify K=1 vs K>1 energy comparison; possibly introduce free energy or model selection criterion
- **Priority 3:** Either adopt stratified Morse theory or abandon smooth Morse framework for M₂ analysis

### For Empiricists
- Continue exp66–exp73 (branch selection dynamics) as orthogonal research; does not resolve F-1
- Investigate actual K emergence in biological/cognitive data (e.g., visual perception, neural oscillations)
- Test whether external K-selection mechanisms exist in nature

## Transparency & Intellectual Honesty

**Philosophy of v1.2:**

This version prioritizes **transparency over polish**. Rather than:
- Hiding F-1/M-1/MO-1 as "implementation details"
- Allowing readers to think multi-formation theory is self-contained
- Pretending unvalidated classifications (Type A/B) are empirical discoveries

We instead:
- Make all three critical problems **explicit and prominent**
- State exactly which theorems depend on which external assumptions
- Retract the Type A/B framework and explain why
- Propose concrete mitigation strategies for each problem

This approach is longer and less "finished-looking," but it's honest about the theory's current state.
