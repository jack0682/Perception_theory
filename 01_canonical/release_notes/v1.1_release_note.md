## K-Field Architecture (New Introduction)

v1.1 introduces the K-field extension framework (multi-formation theory):

### K-Field Configuration
- K independent soft fields on shared relational space
- Simplex constraint: Σ_j u^j_t(x) = 1 everywhere
- Per-formation mass: m_j = ∫ u^j_t dx (externally imposed)
- Global manifold: Σ²_M = {(u¹, u²) : m_1 = m_2 = M/2}

### K-Field Theorems (Provisional, Category B/C)
1. **T-Persist-K-Sep** (Category B): Well-separated formations persist
2. **T-Persist-K-Weak** (Category C): Weakly-interacting formations persist
3. **T-Bind-Proj** (Category B): Bind projects onto lower Σ_m layers (τ=1/2 only)

### Critical Assumption (Not Yet Resolved)
- **Fixed K constraint:** K is externally imposed, not derived from energy minimization
- **Status:** This is a limitation; no mechanism yet for how K emerges naturally
- **Mitigation:** K-field results explicitly state "K fixed" assumption
