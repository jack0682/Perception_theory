- **Gap 4 — Near-bifurcation basin** : μ → 0 근방에서 boundary-mode soft mode가 실제 bottleneck. 방향별 basin bound 미확립. T-Persist-1(b) 조건 재정식화 필요
- **Near-bifurcation persistence** : μ → 0 근방에서 T-Persist-1(b,d) 모두 붕괴. T-Persist-1(a,c) (shifted threshold)만 가능. 별도 국소 이론 필요
- **T-Persist-K-Strong** : Dichotomy (compatible/incompatible overlap) 정의됨, 실험 지지 있음. Morse-theoretic proof 없음 → 추측 단계

- **Strong regime transport fixed-point uniqueness** : Schauder는 존재만 보장. 강한 제도에서 고정점 유일성 미증명
- **exp24 (Basin flow test)** : 미완성. Gradient flow basin이 sublevel-set 추정치보다 큰지 직접 검증 필요
- **Barrier scaling 이론** : Δ_soft ~ 0.037·μ^0.32 실험 확인, 지수 0.32의 이론적 설명 없음

### 낮은 우선순위 / 장기
- **T-Persist-K-Strong Morse 분석** : Σ^K_M 위의 merge/split dynamics 완전 이론화
