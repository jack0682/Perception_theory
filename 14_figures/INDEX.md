# Figures Registry — 14_figures/

Central registry and repository for all publication-ready figures, plots, and visual content.

## Purpose

Organize:
- Source files (Python scripts, data files)
- Exported formats (PDF, PNG for papers)
- Figure specifications (captions, legends, metadata)
- Version tracking (which figure version used in which paper)

## Figure Specifications

Each figure should have:
- **Source code** (reproducible)
- **Data file** (if data-driven)
- **Caption** (concise description)
- **Legend** (axis labels, color scheme)
- **Metadata** (date generated, software version)
