# vision_model_sketch/

**이 디렉토리는 이론 영역이 아니다.**

SCC 이론을 기반으로 한 *가상의* 비전 모델에 대한 아이디어 노트만 보관한다.
이론 작업과 분리되어 있고, 정식 promotion pipeline의 일부가 아니다.

## 위치
- `THEORY/` — 정식 이론 (canonical, working, logs)
- `CODE/` — 실행 가능한 SCC 구현 (scc/, tests/, experiments/)
- **`vision_model_sketch/`** — *응용 아이디어 임시 보관* (이 디렉토리)
- `private_brainstorm/` — 다른 개인 탐색 노트

## 언제 들여다볼 것인가
- 이론이 한 단계 굳어서 응용을 진지하게 검토하고 싶을 때
- 누군가에게 "SCC가 비전에 어떻게 쓰일 수 있나"를 보여줘야 할 때
- 다중-형태 σ 확장(Phase 5) 또는 K 선택(OP-0005)이 해결되었을 때
- 그 외에는 *이론에 집중*

## 우선순위
**이론 > 응용 아이디어.**
이 디렉토리의 어떤 내용도 이론의 진행을 방해해선 안 된다.
응용은 이론이 충분히 굳은 다음에야 의미가 있다.
